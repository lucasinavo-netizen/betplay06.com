Problema #1: La App BetPlay No Abre

Síntomas: tocas el ícono, aparece el logo o pantalla negra, se cierra inmediatamente.

Causas y Soluciones:

  1. APK antiguo incompatible — Android moderno rechaza APKs viejos. Solución: descarga la última versión desde betplay.com.co.
  2. Falta espacio — la app necesita ~200 MB libres para caché. Configuración → Apps → BetPlay → Borrar caché.
  3. Permisos revocados — Configuración → Apps → BetPlay → Permisos → concede todos los solicitados.
  4. Bateria optimizada matando la app — Configuración → Batería → Optimización → BetPlay → "No optimizar".
  5. APK corrupto — desinstala y reinstala desde betplay.com.co.

Importante: si la app te pide conceder permisos extraños (SMS, contactos, galería completa), es APK falso. Desinstala y descarga solo desde betplay.com.co.

Problema #2: No Puedo Iniciar Sesión

Síntomas: la app abre pero no acepta tus credenciales, o login queda cargando infinitamente.

Solución paso a paso:

  1. Verifica conexión — abre cualquier web (google.com) para confirmar que tienes internet.
  2. Verifica credenciales — usa "Olvidé mi contraseña" si hay duda. Cédula/email deben ser exactos.
  3. Borra caché de BetPlay — Configuración → Apps → BetPlay → Borrar caché (NO datos).
  4. Cierra y abre la app — cierre completo (deslizar en recientes), no solo minimizar.
  5. Verifica fecha y hora del teléfono — si está incorrecta, login falla (validación de token). Configuración → Fecha y hora → automático.
  6. Usa web móvil — como alternativa, abre betplay.com.co en Chrome. Si ahí sí funciona, es problema de la app.
  7. Desinstala y reinstala la app.

Problema #3: La App Se Cierra Sola

Síntomas: crash aleatorio, especialmente al cambiar de sección, cargar juegos, o después de unos minutos.

Causas:

  • RAM insuficiente — cierra otras apps, reinicia el teléfono
  • Caché corrupta — Borrar caché de BetPlay
  • Conflicto con otra app — desinstala apps recientes que causaron el problema
  • Versión Android/iOS incompatible — actualiza el OS
  • Juego específico buggy — reporta a soporte

Si solo crashea en juegos específicos: es un bug del proveedor del juego (Pragmatic, Evolution, etc.), no de BetPlay. Reporta con captura y nombre del juego.

Problema #4: La App Va Muy Lenta

Síntomas: carga demora 10+ segundos, animaciones entrecortadas, lag al tocar.

Optimizaciones:

  1. Usa WiFi en vez de datos móviles — si es posible. BetPlay requiere buena conexión para live stream.
  2. Cierra apps en segundo plano — especialmente Chrome, YouTube, WhatsApp que consumen RAM.
  3. Desactiva animaciones Android: Opciones de desarrollador → Escala de animación → 0.5x. Hace todo más rápido.
  4. Libera almacenamiento — elimina archivos antiguos si tu celular tiene menos de 2 GB libres.
  5. Reinicia el celular — suena tonto pero resuelve 60% de los problemas de lentitud.
  6. Actualiza la app — versiones nuevas son más eficientes.

Problema #5-7: Depósitos, Apuestas y Live Stream

#5: No Puedo Depositar

  • Verifica KYC completo — depósitos requieren cuenta verificada
  • Método de pago válido — tarjeta no vencida, Nequi verificado biométricamente
  • Monto mínimo — $10.000 COP para PSE, $5.000 COP Nequi
  • Tu banco bloqueó la transacción — llama a tu banco para autorizar apuestas online

#6: No Puedo Hacer Apuestas

  • Saldo insuficiente — obvio pero común
  • Partido cerrado — ya no se puede apostar (pasó mucho tiempo)
  • Límite de apuesta — BetPlay limita monto por evento
  • Cuenta restringida — verifica en "Mi cuenta" si hay notificaciones

#7: Live Stream No Carga

  • Conexión lenta — live requiere 5+ Mbps estable
  • No disponible en tu región — algunos streams solo para suscriptores BetPlay VIP
  • Saldo requerido — algunos streams solo accesibles con apuesta activa o saldo >$50.000
  • Navegador no compatible — usa Chrome o Safari, no navegadores menos conocidos
